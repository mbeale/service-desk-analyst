# Ivanti Neurons for ITSM

## Overview
Ivanti targets complex enterprise and hybrid infrastructure needs but faces challenges regarding cost and recent security reputation.

## Pricing (2025)
- **Estimated Cost**: ~$80/user/month (Subscription model).
- **Price Hikes (Q1 2025)**:
  - SaaS: +5%
  - Perpetual: +10%
- **Total Cost of Ownership**: Considered high due to mandatory consulting services for deployment.

## Key Features & Updates
- **AI Capabilities**:
  - "Write Assist" for HTML fields.
  - "Email Assist" for generating correspondence.
  - Auto-translation features.
- **User Experience**: Modern Chat UI and Agent Health Monitoring.

## Security
- **Vulnerabilities**: Critical flaw identified in 2025: **CVE-2025-22462** (Authentication Bypass), contributing to a negative security reputation.

## Competitor Analysis
### Weaknesses
- **Complexity**: Difficult to customize; limited search functionality in the self-service portal.
- **Deployment**: Heavy reliance on professional services/consulting.

### SolarWinds SWSD Advantages
- **Architecture**: Cleaner, cloud-first architecture vs. Ivanti's legacy/hybrid complexity.
- **Cost**: Lower barrier to entry and lower TCO.
- **Usability**: Faster deployment and easier configuration.
